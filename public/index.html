Λrchitect

I built Λrchitect as a simpler and quicker option for game composers who are looking to prototype their dynamic audio. Setting up a Wwise or FMOD project just for the sake of prototyping can take awhile, and Λrchitect allows the user to simply upload their tracks, set the type of interactivity, and create a public link for easy shareability.

Examples

Please use the example players below to experiment with Λrchitect's different playback options. You'll find options for basic, looped, layered, branched, crossfaded, and generative playback. Expanded versions of demo tracks can be found on my website, Mosaic Music.
